# notes-app-server
Notes-App-Server using Nodejs

### Functionalities
1. Create Notes
2. Update Notes
3. Delete Notes
4. Read Notes
5. Login User
6. SignUp User
7. Authenticaiton using jwt

### Libraries Used
1. Mongoose (for mongodb)
2. Express (for routing)
3. dotenv (for environment variables)
4. jwt (for authentication)
5. bcrypt (for hashing)
